# web-starter-gulp 1.0.0 

## Features:

- Node & Bower
- Loaders: CJSX, JSX, ES6, SCSS
- Fluxxor, React
- Icons generators: Sprites & Fonts
- Bourbon, Neat


#### Getting Started

```bash
npm install
bower install
```

#### Run development version:

```bash
npm run dev
```

#### Build production version:

```bash
npm run build
```

#### Run production server:

```bash
npm run start
```
